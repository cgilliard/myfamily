◆ chain_guard()

ChainGuard chain_guard ( ChainConfig *  config)

The chain_guard function is used to push a new HeapAllocator to the top of the stack. This function must only be called once per scope.

Parameters
configThe ChainConfig to push to the top of the stack.
Returns
A ChainGuard which is used to indicate when the HeapAllocator should go out of scope and revert to the previous HeapAllocator.

int chain_realloc ( FatPtr *  dst,
FatPtr *  src,
u64  size

Reallocates a FatPtr of the specified size and stores the resulting FatPtr in dst on success using the underlying HeapAllocator at the top of the stack. All data in src is copied to dst's data location.

Parameters
dstThe new FatPtr to allocate and copy data to.
srcThe existing FatPtr to reallocate.
sizeThe new desired size of the reallocated FatPtr.
Returns
0 on success, otherwise returns -1 with appropriate errno value set.
