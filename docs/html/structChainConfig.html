Detailed Description

The ChainConfig specifies which HeapAllocator will be used and whether calls to chain_malloc, chain_realloc, and chain_free should be performed in a synchronous manner using locks.

Parameters
haThe HeapAllocator to push to the top of the stack.
is_syncWhether or not synchronous locking should be used.
